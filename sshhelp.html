If you are getting

      Unable to negotiate with ww.xx.yy.zz port 22: no matching key exchange method found. Their offer: diffie-hellman-group1-sha1

make a host entry and put this in there

  Host ASA2
  	HostName 192.168.55.5
  	HostKeyAlgorithms=+ssh-dss
  	KexAlgorithms diffie-hellman-group1-sha1
  	User admin
